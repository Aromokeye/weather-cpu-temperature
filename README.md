## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/Aromokeye/weather-cpu-temperature

# Access
$ cd weather-cpu-temperature

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
